# Dataset

Dataset yang digunakan dalam proyek ini adalah **PlantVillage Dataset**.

Sumber dataset:
https://www.kaggle.com/datasets/emmarex/plantdisease

Dataset berisi citra daun tanaman sehat maupun yang terinfeksi berbagai jenis penyakit dan digunakan sebagai data klasifikasi pada proyek ini.

> Catatan:
> Dataset tidak bisa upload di repository GitHub karena ukuran file cukup besar. Dataset dapat diunduh melalui tautan di atas.
