## Deskripsi Proyek

Proyek ini dibuat sebagai tugas UAS mata kuliah Kecerdasan Buatan. Tujuan dari proyek ini adalah membangun model machine learning untuk memprediksi penyakit pada tanaman menggunakan algoritma Decision Tree dan K-Nearest Neighbor (KNN). Dataset yang digunakan berasal dari PlantVillage yang diperoleh melalui Kaggle.

## Dataset

- Nama Dataset : PlantVillage Dataset
- Sumber : Kaggle
- Link : https://www.kaggle.com/datasets/emmarex/plantdisease

## Algoritma yang Digunakan

- Decision Tree
- K-Nearest Neighbor (KNN)

## Langkah Pengerjaan

1. Mengumpulkan dataset.
2. Melakukan pengecekan dan pembersihan data.
3. Melakukan Exploratory Data Analysis (EDA).
4. Menyiapkan data untuk proses pelatihan.
5. Membangun model Decision Tree dan KNN.
6. Membandingkan hasil kedua model.
7. Melakukan evaluasi menggunakan Accuracy, Precision, Recall, F1-Score, dan Confusion Matrix.
8. Menarik kesimpulan berdasarkan hasil pengujian.
